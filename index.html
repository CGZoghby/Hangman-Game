A Javascript Game

It's a star wars themed hangman game

I shamelessley lifted the (pure css!) scrolling text from:

sitepoint.com/
css3-starwars-scrolling-text/

and then figured out how to set a timer using jQuery so that the game would display after the scrolling text was finished.

I hope you have fun, this game was a blast to make.

A few rules: I liberally used proper nouns (names, planets, etc.) and everything is trained to take your lowercase input. I was not brave enough to figure out making a hangman game process spaces like in between a first an last name, so if the answer is a name, it is only a first name.

Press any (non-alphabet) key to get started!
