Aighto, we are making a hangman game. First things first is break the game of hangman down into concrete steps.

  1. Explain rules of the game
  2. Invite user to play the game by pressing any key
  3. Display a set of blank underlines that are the length of the word the user needs to guess
  4. Invite the user to press any key to guess that character in the word
  5. If successful, fill in the underlines within the word with the correct character they guessed, and display the guessed character
  6. If unsuccessful, display the incorrectly guessed character, and fill in one piece of the hangman
  7. If the user fails and the hangman is hung, refresh the game
  8. If the user guesses the word correctly, add a point to their win tracker, and refresh the game

Welcome to Hangman! Press any key to start playing!
